Cambiar Banco - Cuenta
Cómo probar múltiples bancos:
  1. Cambia algunas cuentas a diferentes bancos usando los botones "Cambiar Banco"
  2. Las transferencias normales solo mostrarán cuentas del mismo banco
  3. Las transferencias interbancarias permitirán enviar a cualquier banco externo
  4. Puedes crear varias cuentas y asignarlas a diferentes bancos para simular un ecosistema completo
